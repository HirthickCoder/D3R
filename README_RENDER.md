# D3R Restaurant App - Render Deployment

This application is deployed on Render with Client ID and Client Key authentication.

## Live URLs

- **Frontend**: https://d3r-frontend.onrender.com
- **Backend**: https://d3r-backend.onrender.com
- **Authentication**: https://d3r-frontend.onrender.com/client-auth

## Authentication

This app uses custom Client ID/Key authentication (not Microsoft login).

Visit `/client-auth` to register and login.
